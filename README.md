# Cohabify

Cohabify is a web app that helps flatmates manage shared living. It makes it easy to split expenses, track chores, and keep everything organized in one place.

---

## 🚀 Features

- Split expenses between flatmates  
- Track and manage chores  
- View dashboard with spending insights  
- Manage members and balances  
- Simple settlement system  

---

## 🛠 Tech Stack

- React (Vite)  
- Tailwind CSS  
- Zustand  
- React Router  

---

## ⚡ How to Run

```bash
npm install
npm run dev
